# Ecom Auto Registration Platform

## Frontend (Vite + React, JavaScript)

```powershell
cd frontend
npm install
npm run dev
```

Note: if you see a Vite/Node engine warning, upgrade Node.js to `22.12+` or `20.19+` (Vite 8+ requirement). This repo pins Vite 5 to work with Node `22.9.0`.

## Backend (Django)

```powershell
cd backend
.\.venv\Scripts\python.exe manage.py runserver
```
